# Scripts-in-Bioinformatics
A collection of useful scripts in bioinformatics

## How to use them
```
#Under Linux
git clone git@github.com:Zhuxitong/Scripts-in-Bioinformatics.git
cd Scripts-in-Bioinformatics
./script -h

#or if you have already git clone
cd Scripts-in-Bioinformatics
git pull
```
